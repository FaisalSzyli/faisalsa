Design Web
and Programming

Desain Pemrograman Web (DPW) adalah proses menciptakan dan membangun situs web yang mencakup desain visual, pengembangan konten, dan pemrograman.

HTML
Foundation of the Web

HTML adalah bahasa markup untuk membuat struktur dasar halaman web, menggunakan elemen seperti header, paragraf, dan gambar.

CSS
Enhancing Web Appearance

CSS digunakan untuk mengatur tampilan halaman web, mengontrol warna, font, tata letak, dan elemen visual lainnya.

JavaScript
Adding Interactivity

JavaScript memungkinkan penambahan elemen interaktif dan dinamis pada halaman web, seperti animasi dan validasi form.

Framework
Tools for Efficient Development

Framework seperti Bootstrap dan React mempercepat pengembangan web dengan menyediakan komponen siap pakai dan pustaka untuk fitur umum.
